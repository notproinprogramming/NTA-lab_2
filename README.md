# NTA-lab_2

# Інструкція користувача для Docker-образу (Лабораторна робота 2)

## 1. Завантаження образу в систему

```
$ docker pull notproinprogramming/nta-dlog
```

## 2. Запуск та використання

Програма приймає аргументи як параметр командного рядка (CLI).
Перший аргумент - номер методу:

- `1` - дискретний логарифм методом перебору (timeout 5 хв)
- `2` - дискретний логарифм алгоритмом СПГ
- `3` - демонстрація обох методів на тестових значеннях (аргументи не потрібні)

Для методів `1` та `2` передаються ще три аргументи: `alpha`, `beta`, `p`.

Програма шукає `x` таке, що `alpha^x = beta (mod p)`.

```
$ docker run --rm notproinprogramming/nta-dlog <номер_методу> <alpha> <beta> <p>
```

### Приклади команд

Перебір:
```
$ docker run --rm notproinprogramming/nta-dlog 1 5 498 503
```

Похліг-Гелман:
```
$ docker run --rm notproinprogramming/nta-dlog 2 2 379 509
```

Демонстрація на тестових значеннях:
```
$ docker run --rm notproinprogramming/nta-dlog 3
```

## 3. Інформування користувача

Якщо запустити контейнер без аргументів або з некоректними аргументами, утиліта автоматично виведе довідку щодо правильного використання. Прапор `--rm` гарантує автоматичне видалення тимчасового контейнера після виведення результату.
